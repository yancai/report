Instance

创建实例

PUT /v1/instance/{release_name}

Describe:
创建服务实例

Request:

Path Parameters:

名称 是否必填 类型 含义
release_name REQUIRED string 名称

Body Parameters:

名称 是否必填 类型 含义
instance_id REQUIRED string 实例id
project_id REQUIRED string 项目id
tenant_id REQUIRED string 租户id
chart_name REQUIRED string chart包名称
configuration OPTIONAL object 配置信息，参考chart包的参数

Response:

http_code:

  • 200
  • 201
  • 202
  • 400
  • 409
???

获取实例信息

GET /v1/instance/{release_name}

Describe:
获取服务实例

Request:

Path Parameters:

名称 是否必填 类型 含义
release_name REQUIRED string 名称

Response:

http_code:

  • 200
  • 410
{
    "type": "string",
    "release_name": "string",
    "chart_name": "string",
    "parameters": {
        "username": "string",
        "password": "string",
        "url": "string"
    }
}

获取实例状态（用于异步）

GET /v1/instance/{release_name}/status

Describe:
获取服务实例状态

Request:

Path Parameters:

名称 是否必填 类型 含义
release_name REQUIRED string 名称

Response:

http_code:

  • 200
  • 410
{
    "status": "string",
    "msg": "string"
}

status枚举值如下：

  • Waiting
  • Running
  • Terminated

Binding

绑定

PUT /v1/binding/{binding_type}/{release_name}

Describe:
绑定

Request:

Path Parameters:

名称 是否必填 类型 含义
binding_type REQUIRED strong 绑定类型，枚举：parameters, syslogDrainUrl, volumeMounts
release_name REQUIRED string 名称

Body Parameters:

名称 是否必填 类型 含义
parameters REQUIRED object 配置信息 
// mysql
parameters = {
    "type": "mysql",
    "configurations": {
        "username": "string",
        "password": "string"
    }
}

// redis
parameters = {
    "type": "redis",
    "configurations": {
        "password": "string"
    }
}

Response:

http_code:

  • 200
  • 201
  • 202
  • 400
  • 409
{
    "type": "string",
    "configurations": {
        "username": "string",
        "password": "string",
        "url": "string"
    }
}
